# BigTire (MindForge) 🛞

BigTire is an AI-driven problem-solving engine that reframes how we approach complex, overwhelming tasks. It is built on a simple physical metaphor: **Pressure = Force / Area** ($P = \frac{F}{A}$).

Instead of acting like a standard AI chatbot that hallucinates immediate answers to impossible questions, the BigTire Cognitive Engine acts as a diagnostic state machine. It takes a heavy objective (Force), identifies the missing context, and asks targeted questions to expand the constraints (Area). As the surface area expands, the perceived complexity (Pressure) drops to zero, revealing a transparent, mechanical solution.

## 🚀 Core Features
* **The Cognitive Engine:** Powered by the Google Gemini API, programmed with strict system instructions to interrogate constraints rather than guess solutions.
* **Dynamic Pressure Gauge:** A real-time UI component that visually tracks the reduction in problem complexity as new context is provided.
* **Force Transparency:** The final application state that generates a comprehensive, step-by-step architectural blueprint once the problem's variables are fully mapped.

## 🛠️ Tech Stack
* **Frontend:** Next.js (React), Tailwind CSS
* **Backend:** Node.js, Express
* **AI Integration:** Google Generative AI (Gemini API)
